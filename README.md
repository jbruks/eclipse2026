# Eclipse 2026

This repository contains all the work related to our observation of the **2026 Total Solar Eclipse** which will be observed from **Mora de Rubielos (Teruel, Spain)**.

## Project Goals

Our objective is to prepare a complete observation campaign, including:

- Planning and documenting both eclipse events.
- Preparing and testing our observation equipment.
- Developing software and automation tools.
- Recording observations, images, and videos.
- Publishing documentation and results.
- Broadcasting the events live over the Internet.

## Observation Site

The observations will take place from our own property in **Mora de Rubielos**, a location chosen for its excellent observing conditions and accessibility.

## Equipment

Our primary optical instrument will be:

- **Sky-Watcher ED100 apochromatic refractor**

Additional equipment will be incorporated into this repository as the project evolves, including mounts, cameras, control software, networking, and power systems.

## Repository Structure

This repository will gradually include documentation and resources for:

- Observation planning
- Astronomical calculations
- Equipment setup and testing
- Telescope control
- Camera configuration
- Image acquisition
- Live streaming infrastructure
- Data processing
- Software development
- Field notes and reports

## Live Streaming

One of the project's main objectives is to provide a live Internet broadcast of both eclipse events, allowing remote viewers to follow the observations in real time.
